Protegendo Agricultores das Queimadas

O projeto Queimada Zero ajuda pequenos agricultores da Amazônia, que são altamente vulneráveis a eventos extremos como queimadas e secas, conforme apontado nos artigos científicos. Utilizando dados de satélite, o sistema envia alertas sobre focos de calor via SMS e WhatsApp, permitindo que os agricultores tomem decisões rápidas para proteger suas lavouras e comunidades — algo essencial em regiões com pouca infraestrutura e sem monitoramento climático local.
Ter alertas é crucial, pois muitos pequenos agricultores não têm acesso a sistemas de monitoramento em tempo real devido à falta de infraestrutura. Isso dificulta o recebimento rápido de informações sobre riscos climáticos, comprometendo suas decisões no campo.
O monitoramento do sistema Queimada Zero é feito utilizando dados de satélite para identificar focos de calor. Com base nesses dados, o sistema envia alertas sobre focos de calor próximos.
Não, você não precisa baixar nenhum aplicativo para receber os alertas. O sistema Queimada Zero foi projetado para enviar alertas via SMS e WhatsApp. Soluções simples, como SMS, são consideradas mais eficazes para comunidades rurais isoladas.
Você pode se cadastrar para receber os alertas do Queimada Zero simplesmente preenchendo o formulário disponível na página Newsletter. Ao se inscrever na newsletter, seu número é incluído na lista de transmissão de alertas climáticos, sem custo e sem complicações!
Você pode conhecer os integrantes envolvidos na concepção e execução do Queimada Zero acessando a página Integrantes!
Além dos alertas, o sistema pode ser expandido com um assistente informativo sobre o clima. Ajudando os agricultores com dicas práticas sobre prevenção de queimadas, ações emergenciais e até estratégias sustentáveis — algo que ainda é uma grande lacuna nas comunidades rurais.
